# Weather Frontend M2

Aplicación frontend de clima desarrollada como MVP utilizando HTML5, Bootstrap y JavaScript.

## Descripción

El proyecto muestra un listado de ciudades de Chile con información climática básica mediante cards responsivas.  
Cada ciudad posee una vista detalle con información ampliada y pronóstico semanal.

---

## Tecnologías utilizadas

- HTML5 semántico
- CSS3
- Bootstrap 5
- JavaScript
- jQuery
- Git y GitHub

---

## Funcionalidades

- Home con 10 localidades de Chile
- Cards responsivas con Bootstrap
- Vista detalle por ciudad
- Pronóstico semanal
- Navegación entre páginas
- Diseño mobile-first
- Manipulación del DOM con JavaScript
- Uso de localStorage
