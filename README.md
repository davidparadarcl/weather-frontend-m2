# Weather Frontend MVP

Aplicación frontend de clima desarrollada con HTML5, Bootstrap y JavaScript.

## Funcionalidades
- Listado de 10 ciudades
- Vista de detalle por ciudad
- Pronóstico semanal (7 días)
- Navegación entre páginas
- Diseño responsivo (mobile-first)

## Tecnologías
- HTML5 semántico
- Bootstrap 5 (grid, componentes)
- JavaScript (DOM + eventos)
- Git/GitHub

## Navegación
- Home: muestra ciudades en cards
- Detalle: información ampliada + pronóstico
- Acerca de: descripción del proyecto

## Cómo ejecutar
1. Clonar o descargar el repositorio
2. Abrir `index.html` en el navegador
